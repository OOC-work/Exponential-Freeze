# Exponential-Freeze-
指數性降溫的炒冰機Exponential cooling ice fryer

# 項目介紹 
有優秀的控制單元和計算單元，來進行運算的一台炒冰機，以卻本溫度變化小，且有多功能控制。

# 項目內容
由硬體+軟體構成，主要以硬體輔軟體，軟體只是用來收發訊號和運算，硬體支撐大部分項目，以超規格來達到良好的結果。  
第一版:Nextion的TTF檔+Arduino的Arduino.ino  
第二版:主要是Arduino的Arduino.ino
# 項目核心
PID算法
